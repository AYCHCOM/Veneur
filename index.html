Veneur is a distributed server implementation of the statsd protocol. It's fast, memory-efficient, and capable of aggregating global metrics.

Veneur is:
Memory-efficient: Perform aggregation in constant space with HyperLogLog and t-digests.
Distributed: Run host-local instances, but also get global aggregate metrics.
Pluggable : Archive data in S3, Redshift, or a custom backend.
Fast: Process even more packets/second than StatsD and DogStatsD.

To get started, check out the manual.
